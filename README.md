Overall idea: Village simulation that can be influenced by livestream viewers.
Aesthetics: 2D Stardew valley, pixly
Features: 
  - Day night cycle
  - Hunger system
  - Cross-villager communication
  - Player assigned NPCs
  - Able to pay for events, like making a specific villager hungry
  - Villagers build stuff
  - Villagers can have kids - allow village to grow
  - Single focus mode, like main character progression, or livestreams from the perspectives of different characters
  - Genetics system
  - Weather system
  - Natural disasters
  - Allow twitchers to control weather and natural disasters
  - Growth/stats for NPC/object representing chatters

Core things/features:
   - Twitch integration/TikTok integration
   - Basic scene
   - NPCs
